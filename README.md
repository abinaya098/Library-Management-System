# Library Management System

Mini Spring Boot REST API project for managing books in-memory.

## Features
- GET all books
- POST add book
- PUT update book
- DELETE delete book

## How to Run
- Java 21
- Spring Boot 3.5.10
- Maven 4.x (or Gradle if used)
- IntelliJ IDEA (or any Java IDE)
- Access API via Postman

## Tools / Dependencies Used
- Spring Boot 3.5.10
- Maven 4.x (for dependency management)
- IntelliJ IDEA (IDE for coding & running project)
- Postman (for testing REST API requests)
- Lombok (optional, for boilerplate code reduction)
- JDK 21

## Notes
- No database used; data is stored in in-memory list
- All operations are REST API based
